Intentando cargar el live timing... Si no aparece, probablemente el servidor no permite iframe o el navegador está bloqueando contenido mixto.

Nota importante

Si esta web está publicada en HTTPS, los navegadores modernos pueden bloquear iframes HTTP por seguridad. En ese caso, aunque el servidor esté online, el visor embebido puede no mostrarse.

La solución real para que funcione perfecto dentro de la página es servir el live timing también por HTTPS o pasarlo por un proxy tuyo.
